A *function* $f$ from a set $X$ to a set $Y$, denoted $f: X \to Y$, assigns each element $x \in X$ exactly one element $f(x) \in Y$. The set $X$ is the *domain* of $f$ and the set $Y$ is the *codomain* of $f$.

The *image* or *range* of $f$, that is $f(X)$, is a subset of $Y$. The range is not necessarily equal to the codomain.

Geometrically, a function satisfies the vertical line test (any constant vertical line cuts through the function at most once).
