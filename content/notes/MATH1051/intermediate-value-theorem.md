---
title: "Intermediate Value Theorem"
tags:
- "MATH1051"
---

Let $f$ be a [continuous function](notes/MATH1051/continuity.md) defined on $[a, b]$, and $c$ some number such that $f(a) < c < f(b)$. Then there exists some $x$ in the domain $[a, b]$ such that $f(x) = c$.

A useful corollary of this theorem is that if a continuous function takes on negative and positive values inside an interval, it must also have a root (i.e. some $x$ in the domain such that $f(x) = 0$).

> 💭 Intuitively, a continuous path between two endpoint values must take on every intermediary value between those two values.
